# hyprcorrect-site

Landing page for [hyprcorrect](https://github.com/jondkinney/hyprcorrect).

The version chip in `index.html` (`<span data-hyprcorrect-version>vX.Y.Z</span>`)
is updated automatically by the main repo's `release-update-site.yml`
workflow on every published Release. Don't remove the `data-` anchor;
copy around it can change.
